# Teste de API - pytest
Projeto demostração de teste de api com pytest

Pré requisitos:
  - Python instalado
  - pip install pytest
  - pip install pytest-reporter

Command Line para executar o teste:
  - python -m pytest -v demo.py --disable-warnings  --html=report\report.html
